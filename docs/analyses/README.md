# docs/analyses

Répertoire de suivi des analyses du projet BIM Spatial Viewer.

## Convention

- Un fichier par analyse, nommé `YYYY-MM-DD_phase-N_[sujet].md`
- Format : l'analyste dépose l'analyse, le dev ajoute ses réponses dans les blocs `> **[Dev]**`
- Les réponses documentent les décisions prises, les correctifs appliqués, et les reports planifiés
